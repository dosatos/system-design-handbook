# Initial Offer Response Scripts

Scripts for responding when you first receive an offer.

---

## Receiving the Offer (Phone Call)

### If Offer is Strong
```
"Thank you so much for this offer - I'm genuinely excited about the
opportunity to join [Company]. I've really enjoyed the interview
process and getting to know the team.

I'd like to take a few days to review the details carefully and
discuss with my family. Could you send me the full offer in writing?
I'd like to have until [specific date, typically 1-2 weeks] to make
my decision.

Is there anything else I should know about the offer that isn't
captured in the written details?"
```

### If Offer is Below Expectations
```
"Thank you for extending this offer - I'm excited about the
opportunity at [Company] and the team is great.

I want to be transparent: the compensation is below what I was
expecting based on my experience and the market data I've seen.
I was targeting a total compensation closer to [X].

Can you share more about how this package was determined? And is
there flexibility to discuss the numbers?"
```

### If Offer is Surprising (Either Direction)
```
"Thank you for the offer. I'm excited about the opportunity.

I want to make sure I fully understand the package. Could you walk
me through the components? [Ask clarifying questions]

I'd like to take some time to review everything carefully. Can you
send the details in writing?"
```

## Following Up After Verbal Offer

### Email: Requesting Written Offer
```
Subject: Thank you - [Company] offer

Hi [Recruiter Name],

Thank you again for the call today and for extending the offer to
join [Company] as [Title]. I'm very excited about this opportunity.

As discussed, could you please send the written offer details?
I'd like to review everything carefully. I'll plan to get back to
you by [date].

If there are any documents or decisions needed from my side before
then, please let me know.

Best,
[Your name]
```

## Asking for Time

### If Pressured for Quick Decision
```
"I understand you need an answer, and I want to give [Company] my
full consideration. Making a career decision this important
deserves careful thought.

I have [other process] wrapping up in the next [X days/week]. Would
it be possible to have until [date] to decide? I want to be fully
committed when I say yes."
```

### If Given Reasonable Deadline
```
"A week sounds reasonable. I'll plan to give you my answer by
[date]. If anything changes on my end, I'll let you know
immediately."
```

### If Deadline is Too Short
```
"I appreciate you need to move quickly. This is an important
decision for both of us, and I want to make sure I'm making the
right choice.

Would [X more days] be possible? I have [reason - other interviews,
family discussion, etc.] that I'd like to complete before deciding."
```

## Clarifying Questions to Ask

### About Base Salary
- "Is there flexibility on the base salary?"
- "How was this base determined relative to the band?"
- "What's the typical raise cycle and percentage?"

### About Equity
- "Can you explain the vesting schedule?"
- "What's the current valuation used for this grant?"
- "What's the refresh grant philosophy?"
- "Is there any flexibility on the number of shares?"

### About Signing Bonus
- "Is there a signing bonus included?"
- "Is there flexibility on the signing bonus?"
- "Are there any clawback provisions?"

### About Level
- "How was this level determined?"
- "What would it take to come in at the next level?"
- "What's the typical timeline to promotion?"

### About the Offer Process
- "What's the typical timeline for decisions?"
- "Is there room to discuss the package?"
- "Who can I speak with if I have questions?"

## Responding to "What Are Your Expectations?"

### If Asked Before Offer
```
"I'm focused on finding the right role and team fit. I understand
[Company] has competitive compensation packages. I'd love to hear
what the range is for this position, and then we can discuss from
there."
```

### If Pushed for a Number
```
"Based on my research and experience, I'm targeting a total
compensation in the range of [range that's 10-20% above your target].
But I'm flexible and more interested in understanding the full
opportunity."
```

### If Asked About Current Compensation
```
"I'd prefer to focus on what's appropriate for this role based on
market rates and my experience. What range does [Company] typically
offer for this position?"
```

Note: In some jurisdictions, companies cannot legally ask about
current compensation. Know your local laws.

## When the Offer Comes Via Email

### Acknowledging Receipt
```
Subject: Re: [Company] Offer

Hi [Recruiter Name],

Thank you for sending over the offer details. I'm excited about the
opportunity to join [Company].

I'm reviewing everything now and may have some questions. Would you
be available for a call on [date/time] to discuss?

Best,
[Your name]
```

### If You Need More Information
```
Subject: Re: [Company] Offer - Questions

Hi [Recruiter Name],

Thank you for the offer! Before I respond, I have a few clarifying
questions:

1. [Specific question about compensation component]
2. [Specific question about equity]
3. [Specific question about other details]

Would it be easier to discuss over a call?

Best,
[Your name]
```

---

## Key Principles

1. **Always express enthusiasm first** - Never lead with concerns
2. **Get it in writing** - Don't negotiate verbal offers
3. **Take your time** - Never accept on the spot
4. **Ask questions** - Clarify before negotiating
5. **Stay professional** - Every interaction matters
