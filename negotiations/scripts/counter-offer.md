# Counter-Offer Scripts

Scripts for negotiating improvements to initial offers.

---

## General Counter-Offer Structure

### The ARIA Framework
1. **Appreciate**: Thank them, express enthusiasm
2. **Research**: Reference market data and your value
3. **Invite**: Open discussion collaboratively
4. **Ask**: Make specific, justified requests

---

## Opening the Negotiation

### Standard Counter-Offer Opening
```
"Thank you again for this offer - I'm very excited about joining
[Company] and working with the team. I've been thinking carefully
about the package, and I'd like to discuss a few components.

Based on my research and my experience with [relevant experience],
I was hoping we could explore [specific area]. Would that be possible?"
```

### Confident Counter-Offer
```
"I really appreciate the offer, and [Company] is my top choice.
After reviewing the details and comparing to market data, I'd like
to propose some adjustments:

For base salary, I'm targeting [$X]. For equity, I was hoping for
[X shares]. Is there flexibility here?"
```

## Component-Specific Scripts

### Negotiating Base Salary

#### If Offer is 10-15% Below Target
```
"The base salary of [$X] is below what I was expecting. Based on my
[X years] of experience, particularly in [relevant area], and the
market data I've researched, I was targeting closer to [$Y].

Is there flexibility to close that gap?"
```

#### If You Have Competing Offer
```
"I want to be transparent - I have another offer with a base salary
of [$X]. I'm more excited about [your company] because [reason],
but the compensation gap is significant.

Can we discuss bringing the base closer to [$Y]?"
```

### Negotiating Equity

#### Standard Equity Ask
```
"I'm excited about the equity opportunity, but I was hoping for a
larger grant. Given my experience and the impact I expect to have,
I was targeting [X shares / $Y value].

Is there flexibility on the equity component?"
```

#### If Equity is Biggest Gap
```
"The base salary looks good, but I'd like to discuss the equity.
Given that equity is a significant part of compensation at [Company],
and my experience in [relevant area], I was hoping for closer to
[X shares].

This would bring the total package more in line with my expectations
and the market."
```

### Negotiating Signing Bonus

#### Standard Signing Bonus Ask
```
"I noticed the package doesn't include a signing bonus. I'm leaving
behind [$X] in unvested equity at my current company, and a signing
bonus would help offset that.

Would a signing bonus of [$Y] be possible?"
```

#### Increasing Existing Signing Bonus
```
"I appreciate the [$X] signing bonus. However, given that I'm
forfeiting [$Y] in unvested equity and my upcoming bonus of [$Z],
I was hoping we could increase it to [$W].

This would help make the transition financially neutral."
```

### Negotiating Level

```
"I'd like to discuss the level for this role. Based on my experience
leading [specific example] and my track record with [achievement],
I believe I'm operating at the [higher level] already.

What would it take to come in at [Level X] instead of [Level Y]?"
```

## Handling Common Responses

### "This is Our Best Offer"
```
"I appreciate you sharing that. I really want to make this work
because [Company] is my top choice.

Could we explore other elements of the package? For example, is
there flexibility on the signing bonus or the equity refresh
schedule? Or perhaps we could discuss a performance-based increase
after my first review?"
```

### "We Can't Match [Competitor]"
```
"I understand. I'm not asking you to match exactly - I'm more
excited about [Company] for [specific reasons].

What I'm hoping for is something that feels fair and makes my
decision easier. If we could get to [$X], I'd be ready to accept
today."
```

### "I'll Need to Check with the Comp Team"
```
"Of course, I understand. I really appreciate you advocating for
this. Is there anything I can provide that would help make the
case - additional context on my experience or competing offers?"
```

### "The Offer is Already at the Top of the Band"
```
"I appreciate you being transparent about that. Given that, a few
questions:

1. What's the timeline for level promotion, which would move me to
   a higher band?
2. Is there flexibility on signing bonus or equity, which might be
   outside the base salary band?
3. What does the refresh grant philosophy look like?"
```

### "We Need an Answer by [Date]"
```
"I understand you need to move forward. I'm very interested in this
role and want to make sure I'm making the right decision.

Can we discuss the package on [earlier date]? If we can align on
compensation, I'm confident I can give you an answer by your
deadline."
```

## Complete Counter-Offer Email

### Email Template
```
Subject: [Company] Offer Discussion

Hi [Recruiter Name],

Thank you again for the offer to join [Company] as [Title]. I've
been giving this careful thought, and I'm very excited about the
opportunity.

After reviewing the package and doing market research, I'd like to
discuss a few components:

1. Base Salary: I was hoping for [$X], which aligns with [reason -
   market data, experience, competing offer].

2. Equity: Given [my experience / impact I'll bring / market data],
   I was targeting [X shares/$Y value].

3. Signing Bonus: With [$X] in unvested equity I'm leaving behind,
   a signing bonus of [$Y] would help make the transition.

I want to emphasize that [Company] is my top choice, and I'm eager
to join the team. I hope we can find a package that works for both
of us.

Would you be available for a call to discuss?

Best,
[Your name]
```

## Final Push Scripts

### When You're Close to Agreement
```
"I really appreciate you working with me on this. I think we're
close. If we can get [specific final ask], I'm ready to sign today."
```

### When You Have a Deadline from Another Company
```
"I want to be transparent with you - I have a decision deadline with
[other company] on [date]. [Your company] is my first choice, but I
need to make a decision.

If we can align on [$X total], I'll commit now. Otherwise, I may
need to make a difficult decision."
```

### Accepting After Successful Negotiation
```
"Thank you so much for working with me on this. I'm thrilled to
accept the offer and join [Company].

Please send over the final paperwork, and let me know what I need
to do for next steps. I'm looking forward to starting on [date]."
```

---

## Key Principles

1. **Always justify your asks** - Market data, experience, competing offers
2. **Stay collaborative** - "How can we make this work?" not "Give me more"
3. **Be specific** - Exact numbers, not ranges
4. **Know your BATNA** - Best Alternative to Negotiated Agreement
5. **Be willing to walk away** - But only if you mean it
