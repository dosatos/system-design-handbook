## Key Patterns to Master

### Tree Traversal Patterns
1. **DFS**: Pre-order, In-order, Post-order
2. **BFS**: Level-order traversal
3. **Path problems**: Track path sum, path to node

### BST Patterns
1. **In-order gives sorted**: Use for kth element
2. **Valid BST**: Track min/max bounds
3. **LCA**: Use BST property or general tree approach

### Trie Patterns
1. **Prefix search**: Navigate to prefix end
2. **Word search**: DFS with backtracking
3. **Autocomplete**: Return all children from prefix

### Heap Patterns
1. **Top K**: Min heap of size K
2. **Merge streams**: Min heap with pointers
3. **Median**: Two heaps (max and min)
