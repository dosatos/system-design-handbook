## Key Topics This Week

### Intervals Pattern
- Sort by start time
- Track end of merged interval
- Detect overlap: start2 <= end1

### Bit Manipulation
- XOR for finding single element
- n & (n-1) removes lowest set bit
- n & -n isolates lowest set bit

### Inference Optimization
- **KV Cache**: Store key/value matrices for past tokens
- **Quantization**: Reduce precision (FP32 → INT8)
- **Speculative Decoding**: Draft with small model, verify with large
- **Batching**: Continuous batching for throughput
