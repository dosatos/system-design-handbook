## Key Patterns to Master

### Stack Patterns
1. **Valid parentheses**: Push opening, pop and match closing
2. **Monotonic stack**: Maintain increasing/decreasing order
3. **Expression evaluation**: Operands and operators

### Binary Search Patterns
1. **Standard**: Find exact element
2. **Left bound**: Find first occurrence
3. **Right bound**: Find last occurrence
4. **On answer**: Binary search the solution space

### Linked List Patterns
1. **Two pointers**: Fast/slow for cycle detection
2. **Dummy head**: Simplifies edge cases
3. **Reversal**: Iterative and recursive
